## 2. Зчитування рядків з файла мовою Python

У другому завданні було реалізовано програмний код мовою **Python**, який забезпечує зчитування рядків з текстового файла. Для роботи з файлом використовується стандартна функція `open()`. Файл відкривається в режимі читання, після чого зчитування даних здійснюється построково за допомогою циклу.

Для безпечної роботи з файлами застосовано конструкцію `with`, яка автоматично закриває файл після завершення виконання програми. Кожен рядок файла обробляється окремо та виводиться на екран.

```python
with open("data.txt", "r", encoding="utf-8") as file:
    for line in file:
        print(line.strip())
