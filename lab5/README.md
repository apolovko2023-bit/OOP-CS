### Виконання роботи
* Результати виконання завдання:
    
    1. **Створення базового класу та порівняння з простими даними:**
        - Створили клас `MyAnimals` з конструктором `__init__`, який приймає параметри: `animals`, `name`, `age` та опціональний параметр `mass`.
        - Порівняли роботу зі звичайними списками та об'єктами класу.
        - Створили три екземпляри класу (`a1`, `a2`, `a3`) та змінили їхні імена через атрибути.
        - Використали метод `__dict__` для виведення всіх атрибутів об'єкта.

    2. **Методи класу:**
        - Реалізували метод `speak()`, який повертає різні звуки для різних тварин (Кіт - "Мяууу", Собака - "Гавв", інші - не розмовляють).
        - Створили метод `add_one_year()`, який збільшує вік тварини на 1 рік.
        - Застосували ці методи до всіх об'єктів у циклі через список `my_zoo`.

    3. **Глобальні змінні класу:**
        - Додали змінні класу `total_animals` (лічильник всіх тварин) та `total_animal_types` (множина типів тварин).
        - Створили кілька об'єктів у циклі та спостерігали за збільшенням глобального лічильника.
        - Дослідили різницю між змінними класу та атрибутами екземпляра.

    4. **Статичні методи:**
        - Реалізували статичний метод `call_pet()` за допомогою декоратора `@staticmethod`.
        - Виявили, що статичний метод можна викликати як з класу (`MyAnimals.call_pet()`), так і з об'єкта (`a1.call_pet()`).
        - Порівняли виклик звичайного методу екземпляра з класу та з об'єкта.

    5. **Properties (властивості):**
        - Створили property `live_untill`, яке обчислює рік, до якого проживе тварина.
        - Додали ще одну property `left_to_live`, яке показує скільки років залишилось жити.
        - Продемонстрували, що properties оновлюються автоматично при зміні віку тварини.

    6. **Методи класу (@classmethod):**
        - Реалізували метод класу `from_csv()`, який створює об'єкт з CSV-рядка.
        - Додали перевірку імені на велику першу букву через метод `istitle()` та автоматичне виправлення через `title()`.
        - Використали змінні класу `total_csv` та `total_objects` для підрахунку статистики.

    7. **Приватні атрибути та методи:**
        - Створили приватні атрибути: `_secret_phrase` (слабка інкапсуляція) та `__secret_strong` (сильна інкапсуляція).
        - Реалізували доступ до приватних атрибутів через properties.
        - Створили приватні методи `_secret_method()` та `__do_not_touch_it()`.
        - Дослідили механізм name mangling для сильно приватних атрибутів.

    8. **Магічні методи:**
        - Реалізували `__repr__()` для технічного представлення об'єкта.
        - Створили `__str__()` для зручного виведення об'єкта користувачу.

    9. **Арифметичні магічні методи:**
        - Реалізували метод `__sub__()` для віднімання (тварина кусає іншу тварину, або старіння на певну кількість років).
        - Створили метод `__gt__()` для порівняння віку тварин.
        - Додали метод `__len__()` для визначення довжини імені тварини.

    **Навчились:**
        - Створювати класи з конструктором `__init__` та різними типами параметрів (обов'язкові та опціональні).
        - Працювати зі **змінними класу** (глобальними для всіх екземплярів) та **змінними екземпляра** (унікальними для кожного об'єкта).
        - Використовувати різні типи методів: звичайні методи екземпляра, статичні методи (`@staticmethod`) та методи класу (`@classmethod`).
        - Створювати властивості через декоратор `@property` для обчислюваних атрибутів.
        - Інкапсулювати дані через приватні атрибути (`_attribute` та `__attribute`).
        - Перевизначати магічні методи для зручної роботи з об'єктами (`__repr__`, `__str__`, `__sub__`, `__gt__`, `__len__`).
